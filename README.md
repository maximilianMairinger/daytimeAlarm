# Daytime alarm

Set an (recurring) alarm at a daytime.

> Please not that Daytime alarm is currently under development and not yet suited for production

## Installation

```shell
 $ npm i daytime-alarm
```

## Usage



```ts
import daytimeAlarm from "daytime-alarm"

daytimeAlarm()
```
